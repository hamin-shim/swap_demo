# SWAP Demo PPT Flow

갤럭시 신제품 기능 소개 톤으로 구성한 데모 장표 순서와 문구 초안입니다. 화면은 `demo_captures`에 있는 캡쳐 파일을 기준으로 배치합니다.

## 1. 기능 소개

**사용 화면**
- `gs_01_home.png`

**큰 문구**
결제 직전, 가장 유리한 조합을 바로 제안합니다

**보조 문구**
Samsung Wallet이 현재 위치, 보유 카드, 멤버십, 쿠폰 조건을 함께 확인해 예상 혜택을 먼저 보여줍니다.

**발표 멘트**
사용자가 결제 화면을 열면 SWAP은 현재 매장을 인식하고, 지금 쓸 수 있는 카드와 혜택 조합을 계산합니다. 사용자는 복잡한 카드 조건을 직접 비교하지 않아도 결제 전에 예상 혜택을 확인할 수 있습니다.

## 2. 위치 기반 추천

**사용 화면**
- `gs_01_home.png`
- `gs_03_location_sheet.png`

**큰 문구**
장소가 바뀌면 추천도 달라집니다

**보조 문구**
매장이 잘못 인식된 경우에도 사용자가 직접 위치를 바꾸고, 적용 후 추천 조합을 다시 확인할 수 있습니다.

**발표 멘트**
주유소처럼 혜택 조건이 까다로운 업종에서는 매장과 제휴사가 중요합니다. SWAP은 위치를 기반으로 추천을 시작하되, 사용자가 다른 매장을 선택할 수 있도록 해 추천 결과의 신뢰도를 높였습니다.

**사용 화면**
- `gs_04_combo_sheet.png`

**큰 문구**
주유 할인과 포인트 적립을 한 번에

**보조 문구**

**발표 멘트**

## 4. 기준을 바꾸면 결과도 미리 확인

**사용 화면**
- `gs_05_detail_expanded.png`
- `gs_06_detail_mileage_preview.png`

**큰 문구**
혜택, 마일리지, 실적 기준을 사용자가 직접 선택합니다

**보조 문구**
기준을 누르는 즉시 조합이 어떻게 달라지는지 보여주고, 적용을 눌렀을 때만 실제 추천에 반영됩니다.

**발표 멘트**
모든 사용자가 같은 기준으로 결제하지는 않습니다. 어떤 사용자는 할인 금액을, 어떤 사용자는 항공 마일리지를, 또 어떤 사용자는 카드 실적 충족을 우선합니다. SWAP은 기준 변경 결과를 먼저 보여주고 사용자가 확정하도록 설계했습니다.

## 5. AI 추천 설정

**발표 멘트**
설정 화면에서는 추천 기준을 사용자가 직접 조정할 수 있습니다. 혜택, 마일리지, 실적 기준을 바꿔 보며 추천 결과를 확인하고, 확정한 기준은 이후 Wallet 추천에도 반영됩니다.

## 6. SWAP 결제 1단계

**사용 화면**
- `gs_07_pay_benefit_step.png`
- `gs_08_pay_extra_options.png`

**큰 문구**
결제 전에 필요한 혜택을 먼저 챙깁니다

**보조 문구**
멤버십이나 쿠폰 바코드를 먼저 보여주고, 사용자가 다른 쿠폰이나 멤버십도 선택할 수 있게 합니다.

**발표 멘트**
SWAP은 추천 카드만 보여주는 데서 끝나지 않습니다. 실제 결제 순서에 맞춰 멤버십이나 쿠폰을 먼저 제시하고, 필요하면 다른 혜택도 바로 꺼내 쓸 수 있도록 합니다.

## 7. SWAP 결제 2단계

**사용 화면**
- `gs_09_pay_card_step.png`

**큰 문구**
혜택 적용 후, 바로 카드 결제로 이어집니다

**보조 문구**
바코드 사용이 끝나면 추천 카드가 NFC 결제 화면으로 전환되어 결제 흐름이 끊기지 않습니다.

**발표 멘트**
기존에는 사용자가 멤버십 앱, 쿠폰, 카드 결제를 따로 떠올려야 했습니다. SWAP은 이 순서를 하나의 결제 흐름으로 묶어 사용자가 놓치기 쉬운 혜택을 자연스럽게 챙기도록 돕습니다.

## 8. 결제 결과 확인

**사용 화면**
- `gs_10_payment_result.png`

**큰 문구**
방금 결제로 얼마나 유리해졌는지 보여줍니다

**보조 문구**

**발표 멘트**
결제가 끝나면 사용자는 예상 혜택과 실적 진행 상황을 확인합니다. 실제 청구 할인이나 한도는 카드사 기준에 따라 달라질 수 있기 때문에, 카드사 앱에서 최종 확인할 수 있는 연결도 함께 제공합니다.

**발표 멘트**

## 10. 베스킨라빈스 시나리오 전환

- `baskin_01_home.png`

업종이 바뀌면 혜택 조합도 바뀝니다

**보조 문구**

**발표 멘트**

## 12. 마무리

**사용 화면**
- `gs_01_home.png`
- `baskin_01_home.png`

**큰 문구**
Samsung Wallet이 결제의 다음 행동까지 제안합니다

**보조 문구**
SWAP은 카드 추천을 넘어, 쿠폰, 멤버십, 실적, 결제 순서를 하나의 Wallet 경험으로 연결합니다.

**발표 멘트**
SWAP의 핵심은 단순한 카드 추천이 아니라 결제 직전에 필요한 판단과 행동을 줄이는 것입니다. 사용자는 지금 어디서, 어떤 기준으로, 어떤 순서로 결제하면 좋은지 Wallet 안에서 바로 확인할 수 있습니다.

## 짧은 한 줄 카피 후보

- 결제 직전, Wallet이 가장 유리한 순서를 제안합니다.
- 카드 혜택부터 멤버십 적립까지, 놓치기 쉬운 혜택을 한 흐름으로.
- 할인, 마일리지, 실적. 원하는 기준으로 결제 조합을 바꿔보세요.
- 복잡한 카드 조건은 SWAP이 계산하고, 사용자는 결제만 이어갑니다.
- 추천은 필요할 때만 펼치고, 기본 결제 경험은 그대로 유지합니다.
