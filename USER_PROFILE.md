# SWAP Demo User Profile

작성일: 2026-07-13  
기준 파일: `script.js`, `SUSEO_REAL_LOCATION_SCENARIO_DRAFT.md`

이 문서는 수서 실제 위치 기반 SWAP 데모가 전제하는 메인 사용자를 정의한다.

## 1. 메인 사용자

메인 사용자는 수서동 SK 주유소 근처에서 Samsung Wallet을 여는 혜택 민감형 갤럭시 사용자다. 카드별 할인 조건, 선택 정유사, 실적, 쿠폰, 멤버십을 직접 기억하기 어렵기 때문에 결제 직전에 SWAP 추천을 확인한다.

## 2. 기본 우선순위

| 항목 | 현재 데모 기준 |
| --- | --- |
| 기본 위치 | SK엔크린 중앙에너비스 수서지점 |
| 기본 추천 기준 | 혜택 |
| 기본 추천 카드 | 신한 Deep Oil |
| 기본 조합 | SK 주유쿠폰 확인 -> OK캐쉬백 바코드 제시 -> Deep Oil 결제 |
| 보조 기준 | 마일리지, 실적 |
| 결제 전 원칙 | 실제 승인금액을 가정하지 않음 |
| 결제 후 원칙 | 승인금액 기준으로 할인/적립 계산 |

## 3. 보유 카드

| 순서 | 카드 | 데모상 역할 | 현재 상태 |
| --- | --- | --- | --- |
| 1 | KB국민 주유패스 | 실적 채우기/리터당 할인 후보 | 이번 달 실적 489,000원/500,000원, 혜택 조건까지 11,000원 남음 |
| 2 | 삼성 iD STATION | 과거 주유 제휴형 비교 후보 | 현재 SK 주유소에서는 직접 할인 근거가 약함 |
| 3 | 신한 Deep Oil | 수서 SK 주유소 기본 추천 카드 | 선택 정유사 SK에너지, 전월 실적 충족, 주유 할인 대상 이용금액 92,000원 남음 |
| 4 | 현대카드 M BOOST | 포인트/카페/배스킨 후보 | M포인트 18,400P 보유, 배스킨라빈스 50% 포인트 사용 후보 |

## 4. 멤버십/쿠폰

| 자산 | 데모상 표현 |
| --- | --- |
| SK 주유쿠폰 | 자동 적용이 아니라 `SK 주유쿠폰 확인`으로 표시 |
| OK캐쉬백 | 자동 조회가 아니라 `OK캐쉬백 바코드 제시`로 표시 |
| 해피포인트 | 배스킨라빈스 후보에서 적립 |
| 투썸하트 | 투썸플레이스 후보에서 적립 |
| 현대카드 M포인트 | 배스킨라빈스에서 결제금액 50% 사용 요청 가능 |

## 6. 내부 승인금액

아래 금액은 결제 후 결과 화면 계산용이다. 결제 전 화면에는 노출하지 않는다.

| 위치 | 내부 승인금액 | 사후 결과 |
| --- | --- | --- |
| SK엔크린 중앙에너비스 수서지점 | 45,000원 | 카드할인 4,500원 + OK캐쉬백 46P |
| 배스킨라빈스 강남수서점 | 14,600원 | M포인트 7,300P 사용 + 해피포인트 적립 |
| 투썸플레이스 강남수서점 | 12,800원 | M포인트 128P + 투썸하트 적립 |

## 7. 데이터 전제

뱅크샐러드 협업으로 확인한다고 보는 정보:

- 보유 카드
- 카드 이용내역
- 전월 실적 충족 여부
- 이번 달 실적 진행률
- 카드별 할인 대상 이용금액/잔여 한도
- M포인트 등 포인트 잔액
- 결제 후 승인금액 기반 혜택 계산

조심해서 표현해야 하는 정보:

- 결제 전 실제 결제금액
- 카드사 청구할인 최종 확정액
- 쿠폰 사용 가능 주유소/최소 결제액
- OK캐쉬백 잔액/이용내역 직접 조회
- 사용자가 카드사 앱에서 고른 선택 정유사 정보의 확정성

따라서 UI는 결제 전에는 `가능`, `확인`, `남은 한도` 중심으로 말하고, 결제 후에만 승인금액 기준 금액을 계산한다.
